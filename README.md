# zixing-cli
A simple webpack scaffold / 一个简易的webpack脚手架工具
## Requirements
* Node尽量使用最新版本
* Webpack >= 4.0

## On the command line
### Install
```
npm install zixing-cli -g
```
### Use
```
zixing init <Project Name>
cd <Project Name>
npm install
```
